Система управления персоналом
Этот проект нужен для того, чтобы в удобном формате управлять сотрудниками.
База данных насчитывает 100 000 сотрудников, у каждого есть свой уникальный ID, дата найма, зарплата, свой начальник и должность.
Реализована возможность сортировать и фильтровать результаты таблицы, а также изменять начальника у любого сотрудника.
